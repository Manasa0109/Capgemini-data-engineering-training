## Day 4 – SQL Functions (String, Numeric, Date, NULL & Regex)
### Overview

On Day 4, I focused on learning and practicing different types of SQL built-in functions, including:

String functions

Numeric functions

Date functions

NULL handling functions

Regular Expressions (Regex)

These functions are essential for data cleaning, transformation, and real-world data processing.

### What I Did

Practiced String Functions:

UPPER(), LOWER(), LENGTH(), SUBSTRING(), TRIM()

Worked with Numeric Functions:

ROUND(), CEIL(), FLOOR(), ABS()

Used Date Functions:

CURRENT_DATE, YEAR(), MONTH(), DATEDIFF()

Applied NULL Functions:

COALESCE(), IFNULL()

Explored Regex Functions:

Pattern matching using LIKE and RLIKE

Performed queries for:

Data cleaning

Extracting patterns

Formatting values

### What I Learned
How to manipulate text using string functions

Performing calculations using numeric functions

Extracting and analyzing time-based data using date functions

Handling missing values using NULL functions

Using Regex for pattern matching and filtering data

Importance of functions in data preprocessing

### Challenges Faced

Understanding correct syntax for regex patterns

Handling NULL values correctly in queries

Remembering different functions and their usage

### How I Solved It

Practiced each function with small examples

Compared outputs of similar functions (LIKE vs RLIKE)

Tested queries with different inputs

Repeated practice to remember syntax and usage

### Outcome

Strong understanding of SQL functions for data processing

Improved ability to perform data cleaning and transformation

Prepared for interview questions on SQL functions
