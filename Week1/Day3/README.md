## Day 3 – SQL Window Functions & CASE WHEN

### Overview

On Day 3, I worked on advanced SQL concepts including Window Functions and CASE WHEN statements using an Employee dataset.

The focus was on performing data analysis, ranking, and conditional transformations, which are commonly used in real-world data engineering tasks.

### What I Did

Created and populated the Employee table

Retrieved data using SQL queries

Applied Window Function:

ROW_NUMBER() to rank employees based on salary

Used CASE WHEN to:

Categorize employees based on salary levels

Performed data analysis combining:

Sorting

Ranking

Conditional logic

### What I Learned

How to use ROW_NUMBER() for ranking records

Understanding Window Functions vs GROUP BY

Writing CASE WHEN conditions for data transformation

Applying conditional logic for:

Data categorization (High / Medium / Low)

Combining multiple SQL concepts in a single query

### Challenges Faced
Understanding how window functions process each row

### How I Solved It
Practiced window functions with ordered data

Observed output carefully to understand behavior

Broke queries into smaller steps and tested

### Outcome
Gained strong understanding of:

Window Functions (ROW_NUMBER)

CASE WHEN logic

Improved ability to write analytical SQL queries

Built confidence in solving interview-level problems
